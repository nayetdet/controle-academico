Mini Sistema de Controle Acadêmico

Bem-vindo! Utilize o menu superior para gerenciar alunos, disciplinas e matrículas.

  • CRUD completo para alunos, disciplinas e matrículas.
  • Regras de integridade: matrículas duplicadas são bloqueadas e exclusões respeitam vínculos.
  • Autenticação com Spring Security e perfis ADMIN e SECRETARIA.
